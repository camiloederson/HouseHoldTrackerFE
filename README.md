# HouseholdTracker Frontend

## 📌 Overview
This is the frontend for HouseholdTracker — a Progressive Web App (PWA) built with Angular.  
It provides a responsive interface for managing households, budgets, categories, and expenses.

## 🛠 Tech Stack
- Angular
- Tailwind CSS
- SweetAlert2
- PWA support
- Deployed on Netlify

## ✨ Features
- User login and registration with JWT integration
- Responsive UI for CRUD operations
- SweetAlert2 notifications
- Mobile‑friendly PWA functionality

## 🔮 Future Improvements
- Enhanced UI polish
- Debt control notifications
- Analytics dashboards

## 🚀 Setup
```bash
git clone <repo-url>
cd householdtracker-frontend
npm install
ng serve
